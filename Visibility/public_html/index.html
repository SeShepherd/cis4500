Ghost Hunt

Story:

Welsh the dog has a task of scaring away the ghost. Only a combination of a bright light and a bark will scare a ghost away.Be careful since if the light will
take time to recharge if it's left on The ghost will only be visible if they are in darkness. How many ghost can you help remove before the music stops?

How to Move:

Use the arrow keys to move welsh, the space bar will control on the light source.

(The light source will only stay on for 3 moves, and it takes 3 moves to recharge if completely depleted)
