Rules:

For Players 2-4
Each turn begins with the player either rolling or guessing.
Guessing:
Select the colour and the number you think the ball will be. Then click guess for a ball.
- If the either the color or the number was CORRECT move forward an additional square
- If both the color and the number were CORRECT jump up a row
- If both the color and the number were INCORRECT move backwards 2 squares
Rolling:
Click roll to get a ball. The player will move the indicated number on the ball.

To Win:

You must move the exact number onto the Final square, else you move backwards the amount on the ball.
